Что особенного в clojure?

Clojure это современный диалект lisp

Джон Маккарти

  • американский информатик
  • автор термина "искусственный интелект"
  • в 1958 году изобрел язык lisp
  • основоположник функционального программирования

Особенности lisp

  • first class functions
  • garbage collection
  • dynamic typing
  • homoiconicity
  • macros

Functional programming in clojure

  • Отличительная особенность ФП - это подход работе с изменяемым состоянием
  • Императивная парадигма поощряет изменяемое состояние, используем когда удобно
  • Функциональная парадигма: максимально избегаем изменяемого состояния

Referential transparency

Ссылочная прозрачность - возможность заменить выражение на его результат без изменения поведения программы

Чистые функции

  • без побочных эффектов
  • хорошо тестируются
  • хорошо компонуются
  • легко заменяются

Монады

Монады - абстракция в чистых функциональных языках, инкапсуляция функций с побочными эффектами (например IO в Haskell)

В clojure монад нет

  • разделение чистого кода, от кода с побочными эффектами - на усмотрение программиста

Immutability in clojure

Неизменяемые структуры данные

  • 2 + 2 -> 4
  • [1 2 3] conj 4 -> новый вектор [1 2 3 4]

Structural sharing

Efficient by time and space complexity

Clojure is designed for concurrency

  • Reference types: atom, ref, agent
  • No lock based synchronization

Software transactional memory

  • транзакция - часть кода, которая читает и пишет в разделяемую память
  • чтение и запись логически происходит в один момент времени, промежуточные состояния не видны другим транзакциям
  • транзакция либо выполняется полностью, либо ретраится

https://en.wikipedia.org/wiki/Software_transactional_memory

core.async

Go blocks and channels with core.async lib

Clojure is hosted on jvm

  • Java interop and vice versa
  • Unlimited access to all java libraries
  • Compile to bytecode
  • Provided as a clojure.jar (just put it in classpath to start)
  • The same build and deploy process (jar, war)

Базовые типы

  • Arbitrary precision integer - 1231412
  • Doubles - 1.234, BigDecimals - 1.234M
  • Ratios - 22/7
  • Strings - "string", Characters - \a \b \c
  • Symbols - something, Keywords - :key1 :key2
  • Booleans - true false, Null - nil

Структуры данных

  • Список (1 3 "asdf" + \g)
  • Вектор [1 2 3 "zc"]
  • Мап {:key 1 :key2 "val"}
  • Сет #{1 2 3 "asdf"}
  • Любая вложенность

Что такое clojurescript?

  • компилятор clojure -> js, написанный на clojure

Плюсы ClojureScript

  1. все приложение оптимизируется и минифицируется при помощи Google Closure Compiler

    • minification & more
  2. модульность, персистентные структуры данных, иммутабельность из коробки
  3. очень выразительный
  4. repl
  5. macros
  6. core.async

React + ClojureScript
